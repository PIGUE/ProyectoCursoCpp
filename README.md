# ProyectoCursoCpp
# Ejercicios de C++ - Cisco Networking Academy

Este repositorio contiene mis ejercicios personales de aprendizaje en C++ como parte del curso **Cisco Networking Academy: Fundamentos de C++ 1**.

## 📚 Propósito

Desarrollar habilidades en programación C++ mediante prácticas guiadas por el curso oficial. Los ejercicios incluyen:

- Variables y operadores
- Control de flujo
- Entrada y salida
- Funciones y punteros
- Manejo de excepciones

## ⚠️ Nota importante

Este repositorio **no contiene soluciones oficiales** ni tareas evaluadas. Todo el contenido es de carácter educativo y personal.
